## 🔨使用方法

1. 首先，你的系统需要满足以下条件：

    * Windows 7 或更高版本，**不支持XP**。

2. 使用本程序前，先关闭安全软件。

3. **以管理员身份运行本程序**，等待程序获取最新的补丁信息。

4. 输入`.\GDDoS -h`查看帮助

## 👀最佳配置

```
.\GDDoS -m HEAD -ims 1 -cc 10000 -dm 15
```
 解析：
 - 使用HEAD快速请求
 - 使用1ms一次的请求（每秒1000次）
 - 使用10000个线程
 - 攻击15分钟（可选）
