### You can download the compiled program here:

 - [Linux]
 - 我们暂不支持 MacOS 系列系统

### 最佳配置

```
go run GDDoS.go -m HEAD -ims 1 -cc 10000 -dm 15
```
 解析：
 - 使用HEAD快速请求
 - 使用1ms一次的请求（每秒1000次）
 - 使用10000个线程
 - 攻击15分钟（可选）
